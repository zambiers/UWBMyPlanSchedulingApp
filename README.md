### Before running this project Double check this from this list?

* Did you sync to the most recent push?
* is ```npm install react-router-dom``` installed?
* is ```npm install postgres``` installed?
* is ``` npm install @supabase/supabase-js``` installed?
* are you cd into the correct file?

### How to run
```
cd uwb-planner
```
then 

```
npm start
```
